# Screenshots da documentação

Este diretório deve receber capturas reais do sistema. Não use mockups ou imagens
geradas como evidência de funcionalidades implementadas.

| Arquivo esperado | Conteúdo recomendado | Uso principal |
|---|---|---|
| `catalogo.png` | Grade com pesquisa, filtros e cards. | README e funcionalidades. |
| `detalhes-desenho.png` | Modal com preview, variações e ações. | Funcionalidades. |
| `importacao.png` | Seleção ou relatório de importação. | Funcionalidades e desktop. |
| `vitrine.png` | Página pública em viewport de celular. | README e funcionalidades. |
| `app-desktop.png` | Aplicativo Tauri executando no Windows. | README e desktop. |
| `envio-maquina.png` | Escolha da unidade e confirmação de envio. | Desktop. |

## Orientações

- capture dados fictícios ou anonimizados;
- não mostre caminhos absolutos, nomes pessoais, tokens ou domínios internos;
- não inclua consoles com credenciais ou variáveis de ambiente;
- prefira PNG legível e com tamanho razoável;
- atualize a imagem quando a interface mudar de forma relevante.

As imagens só devem ser referenciadas no README depois que os arquivos reais
forem adicionados.
